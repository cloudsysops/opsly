## To stop generation

1. Disable workflow: `.github/workflows/sync-docs.yml` and `.github/workflows/sync-all.yml`
2. Delete this directory
3. Remove scripts: `scripts/sync-docs.js`, `scripts/generate-sprint-tracking.js`, `scripts/test-sync-docs.sh`

See `docs/adr/ADR-033-docs-canonicalization.md` for context.
